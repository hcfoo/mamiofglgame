# Mami of GL

A simple web game prototype built with Next.js App Router.

## Pages
- `/` Home
- `/how-to-play` How to Play
- `/play` Prototype play screen
- `/end` End summary
- `/collection` Zodiac card collection grid

## Deploy on Vercel
1. Create a new GitHub repo and upload these files
2. In Vercel, import the repo and deploy
3. Framework: Next.js (auto-detected)
